# **App Name**: Prompt Master Challenge

## Core Features:

- User Onboarding: User authentication via data privacy agreement and screen name creation.
- Persona Selection: Persona selection with predefined roles and a randomized "surprise me" option.
- Scenario Presentation: Presents scenario-based prompt challenges for selected persona (10 challenges for each persona).
- Prompt Submission: Input form for user to craft and submit prompts in the specified format: Persona, Context, Task, and Format.
- AI Prompt Scoring: Leverages Gemini to assess prompts based on format (Persona, Context, Task, Format) and effectiveness as a tool, generating scores and providing feedback.
- Results Display: Display submitted prompt, score breakdown, and recommended prompt structure for improvement.
- Real-time Leaderboard: Real-time leaderboard showcasing user rankings based on prompt scores.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4), reflecting trustworthiness and technological expertise.
- Background color: Light blue (#E3F2FD), creating a clean and professional feel.
- Accent color: Yellow (#FFEB3B), used sparingly for highlights and call-to-action elements, drawing attention to important interactive areas.
- Body and headline font: 'Inter', a grotesque sans-serif with a neutral, modern look for readability.
- Use professional icons representing different personas and challenge categories.
- Clean, structured layout with clear visual hierarchy to aid comprehension.
- Subtle animations for score updates and leaderboard transitions.
